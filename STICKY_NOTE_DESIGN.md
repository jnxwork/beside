# Sticky Note（便签）功能设计

> 核心定位：不是社交功能，是"空间有记忆"。便签留给空间，不是留给某个人。
> 情感内核："I'm not alone" — 上一个人在这里待过，留下了一句话。

---

## 触发时机：离开时

走到传送门准备离开房间时，底部轻轻浮现：

> *Leave a note?* 📝

- 点了才展开写，不点直接走，零打扰
- 冷却条件：在房间待超过 10 分钟才出现提示
- 为什么是离开时：离开是天然的"回望"时刻，坐下时心思在做事上

---

## 写便签 UI

```
┌─────────────────────────────┐
│  📝 Leave a note            │
│                             │
│  [____________________]     │  ← 限 30 字
│                             │
│  😊 😴 💪 🌧 ☀️    (可选)   │  ← 贴一个心情 emoji
│                             │
│       [Leave]  [Skip]       │
└─────────────────────────────┘
```

- 文字限 30 字，逼迫精练
- 可以只贴 emoji 不写字（降低门槛）
- 也可以只写字不贴 emoji
- 不署名 — 便签上没有名字，只有文字和时间（"2 hours ago"）

---

## 便签位置

房间里固定一个位置：桌子上 / 窗台边 / 墙上软木板

- 不跟随玩家位置，固定在有氛围的家具旁
- 视觉：一张小纸片 sprite，微微歪斜，有手写感
- 没有便签时位置空着，有便签时多出一张小纸
- **同时只存在最近 1 张** — 被新便签替换

为什么只留 1 张：
- 堆叠就变成留言板，和 bulletin board 重叠
- 1 张 = "一个人对下一个人说的话"，是接力，不是广播
- 量小，审核压力也小

---

## 发现和阅读

**发现：** 进入房间时，如果有便签，小纸片微微晃一下（不弹窗不打断）

**阅读：** 走近 / 点击小纸片 → 轻轻展开：

```
┌─────────────────────────────┐
│                             │
│  "今天辛苦了，早点休息"       │
│                         💪  │
│              — 3 hours ago  │
│                             │
│    [❤️]  [🫂]  [😊]  [☕]   │  ← emoji 回应
│                             │
└─────────────────────────────┘
```

- 回应 emoji 出现在纸条角落，像贴纸
- 每人只能贴一个回应（防刷）
- 回应匿名，只看到种类和数量
- 已有 3 个 ❤️ 和 1 个 ☕ → 你知道 4 个人读过这张纸条

---

## 生命周期

```
留便签 → 下一个人看到 → 可回应 → 被新便签替换 / 24h 后褪色消失
```

- 24h 无新便签 → 纸片颜色变淡 → 消失 → 桌面又空了
- 有人留新的 → 直接替换旧的（旧的不存档）

---

## 猫咪联动

猫咪偶尔走到便签旁坐下，歪头看。纯装饰行为，暗示"猫咪也在读"。

---

## 隐私和安全

- 不署名，不关联身份
- 30 字限制天然降低恶意内容空间
- 只保留 1 张，不积累
- 可加简单关键词过滤
- 便签只在房间内可见，不可外部分享

---

## 信息流总结

```
在房间待超过 10 分钟
  ↓
走向传送门准备离开
  ↓
底部浮现 "Leave a note?" 📝
  ↓
（可忽略）或点击 → 写 30 字 + 可选 emoji
  ↓
便签贴在桌上/窗台
  ↓
下一个人进来 → 小纸片微微晃动
  ↓
走近 / 点击 → 读到内容
  ↓
可以贴一个 emoji 回应
  ↓
24h 后褪色 / 被下一张替换
```
