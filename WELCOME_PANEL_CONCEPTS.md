# Welcome Panel 氛围设计方案

> 核心思路：把注册/设置流程包装成一个有仪式感的物件，让玩家觉得自己正在"进入"一个真实的小世界。

---

## 方案 A：居民证 / Resident Card

**概念**：玩家获得一张"Beside 居民证"，填写信息就像在办理入住手续。

```
┌─────────────────────────────────────────────────┐
│  ╔══════════════════════════════════════════╗    │
│  ║  🏠 BESIDE — RESIDENT CARD              ║    │
│  ╚══════════════════════════════════════════╝    │
│                                                  │
│  ┌──────────┐   NAME: ___________________       │
│  │          │   BIO:  ___________________       │
│  │  [角色]  │   LANG: ☑ EN ☐ 中文 ☐ 日本語    │
│  │  预览    │   TIME: 🌅 Morning                │
│  │          │                                    │
│  └──────────┘   NO. #00427                       │
│                  ISSUED: 2024.03.19              │
│                                                  │
│  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─    │
│  [Presets]  [Custom]                             │
│  [Body] [Eyes] [Outfit] [Hair] [Acc]             │
│  ┌──┬──┬──┬──┬──┬──┬──┐                         │
│  │  │  │  │  │  │  │  │  (选项格子)             │
│  └──┴──┴──┴──┴──┴──┴──┘                         │
│                                                  │
│           [ 🖋 That's me! ]                      │
│        or Login / Register                       │
└─────────────────────────────────────────────────┘
```

**视觉元素**：
- 外框：淡奶白卡片，细虚线边框（像真实证件的裁切线）
- 左上角：小 logo + "RESIDENT CARD" 标题，像证件抬头
- 角色预览区有浅色照片框边框（像证件照位置）
- 底部有编号 `NO. #00427`（可以用实际注册顺序）和签发日期
- 虚线分割线把"证件信息"和"形象定制"分开
- 按钮文字改为 🖋 笔的图标 + "That's me!"（像签名确认）
- 整体色调：暖白底 + 深蓝灰文字，像复古公文纸

**氛围感**：你正在成为这个小社区的居民，这是你的身份证明。

## 方案 B：车票 / Arrival Ticket

**概念**：玩家拿到一张开往 Beside 的车票，填写信息就像在购票。

```
┌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌┐
╎  ══ BESIDE LINE ══          ☆ ONE WAY ☆        ╎
╎                                                  ╎
╎  PASSENGER ___________________                   ╎
╎                                                  ╎
╎  ┌──────────┐  MEMO ___________________         ╎
╎  │  [角色]  │  LANG ☑ EN ☐ 中文                 ╎
╎  │  预览    │  DEPART 🌙 Night                   ╎
╎  └──────────┘                                    ╎
╎                                                  ╎
╎  ┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄ ✂ ┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄┄  ╎
╎                                                  ╎
╎  [形象定制区]                                    ╎
╎  [Presets] [Custom]                              ╎
╎  ...                                             ╎
╎                                                  ╎
╎  PLATFORM 3 · SEAT 27A     [ 🚂 Board! ]       ╎
╎  Valid: 2024.03.19 ~                             ╎
└╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌╌┘
```

**视觉元素**：
- 外框：虚线边框（像票据裁切线）
- 顶部：线路名 "BESIDE LINE" + "ONE WAY" 徽章
- 字段标签用票据风格：PASSENGER / MEMO / LANG / DEPART
- 中间剪刀虚线 ✂ 分割上下两部分
- 底部随机生成站台号和座位号（纯装饰趣味）
- 按钮文字 "Board!" 或 "上车！"
- 色调：淡黄/米色底（像老式车票），红色或深蓝 accent

**氛围感**：一段新旅程即将开始，你要登上开往陪伴空间的列车。

## 方案 C：信件 / Welcome Letter

**概念**：收到一封来自 Beside 的邀请信，填写信息就像回信。

```
┌─────────────────────────────────────────────────┐
│                                                  │
│            ✉ Welcome to Beside                   │
│     ─────────────────────────────                │
│                                                  │
│  Dear new friend,                                │
│                                                  │
│  We've been waiting for you.                     │
│  Please tell us a little about yourself:         │
│                                                  │
│  I go by: ___________________                    │
│  About me: ___________________                   │
│                                                  │
│  ┌──────────┐  I speak: ☑ EN ☐ 中文             │
│  │  [角色]  │  It's now: 🌅 Morning here        │
│  │  预览    │                                    │
│  └──────────┘                                    │
│                                                  │
│  ┈┈┈ Choose your look ┈┈┈                        │
│  [形象定制区]                                    │
│                                                  │
│  Yours truly,                                    │
│           [ ✍ That's me! ]                       │
│                                                  │
│  P.S. You can change everything in ⚙️ anytime.  │
│       or Login / Register                        │
│                                                  │
└─────────────────────────────────────────────────┘
```

**视觉元素**：
- 外框：信纸质感（可加微弱横线纹理 via CSS repeating-linear-gradient）
- 顶部信封图标 + 标题
- 引导文案用书信语气："Dear new friend, We've been waiting for you."
- 字段标签更亲切："I go by" / "About me" / "I speak"
- 底部用书信结尾："Yours truly" + 签名按钮
- P.S. 小字提示可以修改
- 色调：暖白底 + 手写风格字体 accent（标题用），内容仍用像素字体
- 可加半透明邮戳装饰（圆形 "BESIDE · EST. 2024"）

**氛围感**：你收到了来自新世界的一封温暖的信，邀请你加入。

## 方案 D：冒险者手册 / Adventurer's Journal

**概念**：翻开一本手册的第一页，登记自己的冒险者信息。

```
┌─────────────────────────────────────────────────┐
│ ┌───┐                                            │
│ │ ☆ │  ADVENTURER'S JOURNAL                      │
│ └───┘  ━━━━━━━━━━━━━━━━━━━━                      │
│                                                  │
│  « Page 1 — About You »                         │
│                                                  │
│  ┌──────────┐  Name ___________________          │
│  │  [角色]  │  Motto ___________________         │
│  │  预览    │  Languages ☑ EN ☐ 中文             │
│  │          │  Local Time 🌙 Night               │
│  └──────────┘                                    │
│                                                  │
│  « Page 2 — Your Look »                         │
│  ─────────────────────                           │
│  [形象定制区]                                    │
│                                                  │
│          [ ✦ Begin Adventure ]                   │
│  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─                   │
│  Already have an account? Login / Register       │
└─────────────────────────────────────────────────┘
```

**视觉元素**：
- 外框：书本页面感，左侧可加装订线（竖线 + 小圆点）
- 章节标记 « Page 1 » « Page 2 » 分隔信息和形象定制
- 标题区有小星星/徽章图标
- 底部按钮 "Begin Adventure" / "开始冒险"
- 色调：羊皮纸暖黄底 / 深棕文字（RPG 手册风）
- 可加轻微纸张纹理

**氛围感**：你是一个新的冒险者，正在登记自己的信息，准备出发。

## 我的推荐

**方案 A（居民证）** 最适合 Beside：
1. 和"陪伴空间"的定位最匹配——你是来定居的，不是路过的
2. 证件风格天然有表单布局，和现有的输入框/选项结构完美契合
3. 编号/日期等小装饰增加真实感和收藏感，但不会过度
4. 不需要大幅改布局，在现有结构上加证件装饰即可
5. "That's me!" 按钮本身就像签名确认，呼应证件主题

**方案 C（信件）** 是第二选择，氛围最温暖，但需要较多文案空间。
